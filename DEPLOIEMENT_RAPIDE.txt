🔗 ÉTAPE 4 : LIER AU DÉPÔT GITHUB
════════════════════════════════════════════════════════════════════════════

   ⚠️  IMPORTANT : Remplacez "VOTRE_USERNAME" par votre nom d'utilisateur GitHub !

   ┌────────────────────────────────────────────────────────────────────┐
   │                                                                    │
   │  # Ajouter le dépôt distant                                       │
   │  git remote add origin https://github.com/VOTRE_USERNAME/straven-essential.git
   │                                                                    │
   │  # Renommer la branche en 'main'                                  │
   │  git branch -M main                                               │
   │                                                                    │
   │  # Pousser vers GitHub                                            │
   │  git push -u origin main                                          │
   │                                                                    │
   └────────────────────────────────────────────────────────────────────┘

   📝 GitHub demandera vos identifiants :
      • Username : votre nom d'utilisateur GitHub
      • Password : utilisez un Personal Access Token (PAT)

   🔑 Créer un PAT si vous n'en avez pas :
      1. GitHub → Settings → Developer settings → Personal access tokens
      2. Generate new token (classic)
      3. Permissions : cocher "repo" (toute la section)
      4. Copier le token et l'utiliser comme mot de passe


🌐 ÉTAPE 5 : ACTIVER GITHUB PAGES
════════════════════════════════════════════════════════════════════════════

   1. Aller sur votre dépôt :
      https://github.com/VOTRE_USERNAME/straven-essential

   2. Cliquer sur "Settings" (⚙️ en haut à droite)

   3. Dans le menu de gauche : "Pages"

   4. Configurer :
      ┌────────────────────────────────────────────┐
      │ Source                                     │
      │ ├─ Branch: main                            │
      │ └─ Folder: / (root)                        │
      │                                            │
      │ [Save]                                     │
      └────────────────────────────────────────────┘

   5. ⏱️  Attendre 1-2 minutes pour le déploiement

🔧 PERSONNALISATION POST-DÉPLOIEMENT
════════════════════════════════════════════════════════════════════════════

   1. Mettre à jour README.md :
      - Remplacer "VOTRE_USERNAME" par votre vrai username

   2. Mettre à jour sitemap.xml :
      - Remplacer les URLs par votre vraie URL GitHub Pages

   3. Mettre à jour index.html (Open Graph) :

   4. Créer img/og-image.png (optionnel) :

⚠️  PROBLÈMES COURANTS
════════════════════════════════════════════════════════════════════════════

┌────────────────────────────────────────────────────────────────────────┐
│  ❌ Problème : "Page not found" (404)                                  │
├────────────────────────────────────────────────────────────────────────┤
│  Solutions :                                                           │
│  • Vérifier que GitHub Pages est activé (Settings → Pages)            │
│  • Vérifier que la branche est "main" et le dossier "/" (root)        │
│  • Attendre 2-3 minutes après activation                              │
│  • Vider le cache du navigateur (Ctrl+F5)                             │
└────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────┐
│  ❌ Problème : Git demande le mot de passe en boucle                   │
├────────────────────────────────────────────────────────────────────────┤
│  Solution :                                                            │
│  • Créer un Personal Access Token (PAT) :                             │
│    GitHub → Settings → Developer settings → Tokens                    │
│  • Utiliser le PAT comme mot de passe lors du push                    │
│                                                                        │
│  Ou utiliser SSH :                                                     │
│  git remote set-url origin git@github.com:VOTRE_USERNAME/straven-essential.git
└────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────┐
│  ❌ Problème : Images ne s'affichent pas                               │
├────────────────────────────────────────────────────────────────────────┤
│  Solution :                                                            │
│  • Les chemins ont été corrigés automatiquement                       │
│  • Vérifier dans la console du navigateur (F12)                       │
│  • Les images doivent utiliser des chemins relatifs : img/...         │
└────────────────────────────────────────────────────────────────────────┘

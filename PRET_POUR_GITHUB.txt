📌 MÉTHODE 2 - MANUELLE (ÉTAPE PAR ÉTAPE) :
──────────────────────────────────────────────────────────────────

   1️⃣  Créer le dépôt sur GitHub :
       https://github.com/new
       • Nom : straven-essential
       • Public
       • Ne rien ajouter (README, .gitignore, licence)

   2️⃣  Initialiser Git :
       git init
       git config user.name "Votre Nom"
       git config user.email "votre.email@example.com"

   3️⃣  Premier commit :
       git add .
       git commit -m "🎉 Initial commit - Straven Essential Dashboard"

   4️⃣  Lier au dépôt GitHub :
       git remote add origin https://github.com/VOTRE_USERNAME/straven-essential.git
       git branch -M main
       git push -u origin main

   5️⃣  Activer GitHub Pages :
       • Settings → Pages
       • Source : main
       • Folder : / (root)
       • Save


🔑 AUTHENTIFICATION GITHUB
──────────────────────────────────────────────────────────────────

⚠️  GitHub n'accepte plus les mots de passe classiques !

OPTION A - Personal Access Token (PAT) :
────────────────────────────────────────
   1. https://github.com/settings/tokens
   2. Generate new token (classic)
   3. Permissions : cocher "repo" (toute la section)
   4. Copier le token
   5. Utiliser comme mot de passe lors du push

OPTION B - SSH (recommandé pour usage fréquent) :
──────────────────────────────────────────────────
   1. Générer une clé SSH :
      ssh-keygen -t ed25519 -C "votre.email@example.com"

   2. Ajouter à GitHub :
      cat ~/.ssh/id_ed25519.pub
      → Copier et ajouter sur https://github.com/settings/keys

   3. Changer l'URL remote :
      git remote set-url origin git@github.com:VOTRE_USERNAME/straven-essential.git

   4. Push :
      git push -u origin main

⚠️  PROBLÈMES COURANTS
──────────────────────────────────────────────────────────────────

❌ "Page not found" (404) :
   • Vérifier que GitHub Pages est activé
   • Vérifier la branche (main) et le dossier (/)
   • Attendre 2-3 minutes
   • Vider le cache (Ctrl+F5)

❌ Images ne s'affichent pas :
   • Les chemins ont été corrigés automatiquement
   • Vérifier la console (F12)
   • Tous les chemins doivent être relatifs : img/...

❌ Git demande le mot de passe en boucle :
   • Utiliser un Personal Access Token (PAT)
   • Ou configurer SSH

❌ "remote: Repository not found" :
   • Le dépôt n'existe pas sur GitHub
   • Créer le dépôt d'abord : https://github.com/new
